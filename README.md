# VideoToImages
本Android APP是博客文章[Android: MediaCodec视频文件硬件解码,高效率得到YUV格式帧,快速保存JPEG图片(不使用OpenGL)(附Demo)][blog]的Demo APP。

指定输入视频文件和输出文件夹名，此APP可将视频帧保存为I420、NV21或JPEG格式。
